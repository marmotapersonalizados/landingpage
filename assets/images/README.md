# Marmota image assets

Use estes nomes como pontos de troca no HTML:

- `logo-marmota.png`: imagem usada no logo do header e footer.
- `favicon.png`: favicon e apple touch icon da página.
- `hero-products.png`: composição principal do hero com os produtos da Marmota.
- `product-shirt.jpg`: card de produto, camiseta.
- `product-mug.jpg`: card de produto, caneca.
- `product-bag.jpg`: card de produto, ecobag.
- `product-bottle.jpg`: card de produto, garrafa.
- `kv-mascot-gift.png`: mascote da seção sobre.
- `kv-mascot-heart.png`: mascote do CTA final.

Assets `kv-*` vieram do KV da marca.

## Galerias de produtos

Cada botão "Ver opções" da seção `section-product-grid` abre 9 imagens desta estrutura:

- `gallery/camisetas/01.jpg` até `09.jpg`
- `gallery/canecas/01.jpg` até `09.jpg`
- `gallery/ecobags/01.jpg` até `09.jpg`
- `gallery/garrafas/01.jpg` até `09.jpg`

Para trocar os produtos do popup, substitua os arquivos mantendo o mesmo nome.
